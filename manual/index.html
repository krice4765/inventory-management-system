🏢このシステムについて

このシステムは、会社の商品・在庫・注文・取引先をまとめて管理できるWebサイトです。

📦 商品管理

商品の登録や値段の設定、在庫の最低ラインを決めることができます

📈 在庫管理

今ある商品の数を確認したり、商品の出入りを記録できます

🛒 発注管理

商品を注文したり、少しずつ納品されるときの管理ができます

🤝 取引先管理

商品を仕入れる会社や売る相手の連絡先を管理できます

💡 便利な点: すべての情報は自動で更新されるので、いつでも最新の状態を見ることができます。

🌟 実際の使い方の例

📋 お店を運営する場合の一日の流れ

  1. 朝: ダッシュボードで昨日の売上と在庫不足をチェック
  2. 午前: 在庫が少ない商品があれば取引先に発注
  3. 午後: 商品が届いたら分納入力で在庫を更新
  4. 夕方: 売れた商品を出庫処理で在庫から減らす
  5. 夜: ダッシュボードで今日の結果を確認

🔐ログイン・はじめ方

ログインの手順

🚀 最初にやること

  1. インターネットでこのシステムのサイトを開く
  2. 「ログイン」ボタンを押す
  3. あなたのメールアドレスとパスワードを入力
  4. 「ログイン」ボタンを押して中に入る
⚠️ 大切なこと: パスワードは8文字以上で、数字とアルファベットを混ぜて作ってください。

🔑 パスワードを忘れた時

パスワードの作り直し

  1. ログイン画面で「パスワードを忘れた方」をクリック
  2. メールアドレスを入力
  3. 送られてきたメールのリンクをクリック
  4. 新しいパスワードを設定

安全な機能

  • ✅ 安全なログインシステム

  • ✅ パスワードの作り直しができる

商品を登録する手順

📝 新しい商品を追加する

  1. 左のメニューから「商品管理」をクリック
  2. 「新しい商品を追加」ボタンをクリック
  3. 商品の基本情報を入力（商品名、商品番号、説明）
  4. 値段を設定（仕入れ値段、売値）
  5. 在庫の情報を設定（今ある数、最低限必要な数）
  6. 「保存」ボタンを押して完了

📋 実際の例：「コピー用紙A4」を登録する場合

入力する内容の例

  1. 商品名: コピー用紙A4
  2. 商品番号: COPY-A4-001
  3. 説明: 白色A4サイズのコピー用紙500枚入り
  4. 仕入れ値段: 400円
  5. 売値: 500円
  6. 今ある数: 50箱
  7. 最低限必要な数: 10箱
📋 必ず入力すること: 商品名、商品番号、売値、最低限必要な数は必ず入力してください。

低在庫・過剰在庫のアラート機能

在庫を動かす方法

📦 在庫の増減をする手順

  1. 左のメニューから「在庫管理」をクリック
  2. 動かしたい商品を選ぶ
  3. 「クイック移動」ボタンをクリック
  4. 何をするかを選ぶ（入庫/出庫/移動/調整）
  5. 数と理由を入力
  6. 「実行」ボタンを押して完了

📋 実際の例：商品が売れた時の処理

コピー用紙A4を5箱売った場合

  1. 在庫管理画面で「コピー用紙A4」を探す
  2. その行の「クイック移動」ボタンをクリック
  3. 「出庫」を選ぶ
  4. 数量に「5」と入力
  5. 理由に「販売のため」と入力
  6. 「実行」ボタンを押す
  7. 在庫が50箱から45箱に自動で変わる
🔄 便利な機能: 在庫を動かすと、すぐにダッシュボードの数字も変わります。

注文を作る手順

📝 新しい注文を作る

  1. 「発注管理」→「新規発注」をクリック
  2. 注文先の会社を選ぶ
  3. 注文する商品を追加（商品・数・値段）
  4. いつまでに欲しいかと支払い方法を決める
  5. 「発注書作成」で注文完了
  6. 注文番号が自動でつく

📋 実際の例：コピー用紙100箱を注文する場合

A商事からコピー用紙を注文

  1. 「発注管理」→「新規発注」をクリック
  2. 取引先で「A商事」を選ぶ
  3. 「商品を追加」をクリック
  4. 「コピー用紙A4」を選んで、数量「100」、単価「400」と入力
  5. 納期を「2024年10月1日」に設定
  6. 支払い条件を「月末締め翌月末払い」に設定
  7. 「発注書作成」ボタンを押す
  8. 「PO240920001」のような注文番号ができる

商品が届いた時の処理

📦 少しずつ届く商品の管理

  1. 発注一覧から届いた注文を選ぶ
  2. 「分納入力」ボタンをクリック
  3. 実際に届いた商品と数を確認して入力
  4. 届いた日を設定
  5. 「分納実行」で処理完了
  6. 在庫と進み具合が自動で更新される

📋 実際の例：コピー用紙が30箱だけ先に届いた場合

100箱注文のうち30箱が先に到着

  1. 発注一覧で「PO240920001」を探してクリック
  2. 「分納入力」ボタンをクリック
  3. 商品「コピー用紙A4」の分納数量に「30」と入力
  4. 分納日を今日の日付に設定
  5. 「分納実行」ボタンを押す
  6. 在庫が自動で30箱増える
  7. 注文の進み具合が30%と表示される
✅ 便利な機能: 商品が届くと在庫も自動で増えて、注文がどれくらい進んでいるかも自動で計算されます。

🌟完全な作業例

実際の業務の流れに沿って、システムの使い方を説明します。

🏪 例：文房具店の1日の業務

📅 月曜日の朝：週の始まり

  1. 8:00 システムにログイン
    → メールアドレスとパスワードでログイン
  2. 8:05 ダッシュボードで状況確認
    → 低在庫アラートで「ボールペン」が3本しかないことを発見
    → 今月の売上と進行中の注文数をチェック
  3. 8:10 在庫確認
    → 在庫管理で「ボールペン」の詳細を確認
    → 最低10本必要なのに3本しかない状態
  4. 8:15 緊急発注
    → 発注管理→新規発注で「文具商事」にボールペン50本を注文
    → 納期を「明日」に設定

📅 火曜日の午後：商品到着

  1. 14:00 商品到着
    → ボールペン30本が先に届く（残り20本は明日）
  2. 14:05 分納入力
    → 発注一覧から昨日の注文を選択
    → 分納入力で「30本」と入力
    → 在庫が3本から33本に自動更新
  3. 14:10 売上処理
    → お客様がボールペン5本購入
    → 在庫管理で出庫処理「5本 販売のため」
    → 在庫が33本から28本に変更

📅 水曜日の朝：残り商品到着と新規取引先

  1. 9:00 残り商品到着
    → ボールペン20本が到着
    → 分納入力で「20本」と入力
    → 注文の進捗が100%完了に
  2. 9:30 新しい取引先登録
    → 新しい文具メーカー「ABC文具」と取引開始
    → 取引先管理で会社情報と支払い条件を登録
  3. 10:00 新商品登録
    → ABC文具から新しいマーカーペンが入荷予定
    → 商品管理で「蛍光マーカー黄色」を登録
    → 仕入れ値150円、売値200円、最低在庫5本で設定

🔧困った時の解決方法

よくある問題と解決方法

🔄 ページが読み込まれない

  1. ブラウザのページ更新（F5キー）
  2. ブラウザキャッシュのクリア
  3. インターネット接続確認
  4. ブラウザの再起動

🔐 ログインできない

  1. メールアドレス・パスワードの再確認
  2. Caps Lockの確認
  3. パスワードリセット機能の利用
  4. 管理者への連絡

📊 データが表示されない

  1. 「更新」ボタンをクリック
  2. フィルタ設定の確認
  3. 権限設定の確認
  4. システム管理者への報告
